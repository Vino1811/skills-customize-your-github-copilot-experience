# 📘 Assignment: Data Structures in Python

## 🎯 Objective

Learn how to use and manipulate Python data structures: lists, dictionaries, sets, and tuples. Complete small functions that demonstrate each structure and apply them to simple problems.

## 📝 Tasks

### 🛠️ Lists & Tuples

#### Description
Implement functions that work with lists and tuples: find unique items, convert tuples to lists, and perform basic list transformations.

#### Requirements
Completed program should:

- Implement `get_unique_items(lst)` that returns a list of unique items preserving order.
- Implement `tuple_to_list(t)` that converts a tuple to a list.
- Provide examples or tests in `starter-code.py` demonstrating these functions.


### 🛠️ Dictionaries

#### Description
Work with dictionaries to count frequencies and merge maps.

#### Requirements
Completed program should:

- Implement `count_frequencies(items)` that returns a dictionary mapping items to counts.
- Implement `merge_dicts(d1, d2)` that returns a new dictionary with keys from both; values from `d2` override `d1`.


### 🛠️ Sets

#### Description
Use sets to perform membership tests and basic set operations.

#### Requirements
Completed program should:

- Implement `set_operations(a, b)` that returns a dictionary with `union`, `intersection`, and `difference` results.
- Show examples using sample sets in `starter-code.py`.


## 📁 Files

- `starter-code.py` — starter functions and example usage (provided).
